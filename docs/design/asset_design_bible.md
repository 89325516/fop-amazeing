# Asset Design Bible: Wall Objects & Thematic Elements
# 素材设计圣经：墙体对象与主题元素

This document defines the creative direction for "Wall" assets. In this game, a "Wall" is not just a brick wall, but a thematic obstacle.
本文档定义了“墙”素材的创意方向。在本作中，“墙”不仅仅是砖墙，而是符合主题的障碍物。

## Global Requirements (全局要求)
1. **Perspective (视角)**: Top-down 3/4 view (RPG style). Must distinguish between **Front** (South-facing) and **Top** surfaces clearly.
2. **Orientation (朝向)**: 
   - **Horizontal (2x1, 3x1)**: Detailed front face, visible top.
   - **Vertical (1x2, 1x3)**: Visible side/front perspective. **Cannot just rotate the horizontal image 90 degrees.** Must be drawn as a vertical object.
3. **Animations (动画)**:
   - Required for **2x2** and **3x3** sizes.
   - 4 Frames.
   - Subtle, ambient motion (glowing, swaying, flowing).

---

## 1. Grassland (草原主题)
**Vibe**: Ancient, peaceful, overgrown.
- **1x1**: Mossy Stone Box / Small Bush.
- **2x1 / 3x1 (Horizontal)**: **Fallen Ancient Pillar** (covered in vines) / Long Stone Ruin Wall.
- **1x2 / 1x3 (Vertical)**: **Standing Ancient Pillar** (tall and weathered) / Hedges.
- **2x2 / 3x3 (Big)**: **Ancient Magic Fountain** or **Shrine** (Animated: Water flowing or magical glow).

## 2. Desert (荒漠主题)
**Vibe**: Hot, arid, mysterious.
- **1x1**: Crate / Barrel / Cactus.
- **2x1 / 3x1 (Horizontal)**: **Sandstone Ridge** / Half-buried Sphinx part.
- **1x2 / 1x3 (Vertical)**: **Tall Cactus Cluster** / Obelisk.
- **2x2 / 3x3 (Big)**: **Quicksand Pit** or **Ancient Tomb Entrance** (Animated: Dust swirling or heat haze).

## 3. Ice (冰原主题)
**Vibe**: Cold, sharp, crystalline.
- **1x1**: Ice Cube / Frozen Rock.
- **2x1 / 3x1 (Horizontal)**: **Ice Wall Barrier** (Translucent).
- **1x2 / 1x3 (Vertical)**: **Icicle Spikes** (Rising from ground).
- **2x2 / 3x3 (Big)**: **Giant Cryo-Crystal** (Animated: Pulsing blue light / sparkles).

## 4. Jungle (丛林主题)
**Vibe**: Wild, dangerous, organic.
- **1x1**: Carnivorous Plant (Closed).
- **2x1 / 3x1 (Horizontal)**: **Giant Tree Root** (Horizontal section).
- **1x2 / 1x3 (Vertical)**: **Totem Pole**.
- **2x2 / 3x3 (Big)**: **Ancient Temple Head** or **Giant Rafflesia** (Animated: Swaying petals / blinking eyes).

## 5. Space (太空主题)
**Vibe**: High-tech, metallic, neon.
- **1x1**: Server Node / Cargo Box.
- **2x1 / 3x1 (Horizontal)**: **Control Console** / Pipework conduit.
- **1x2 / 1x3 (Vertical)**: **Server Rack** / Stasis Pod.
- **2x2 / 3x3 (Big)**: **Reactor Core** (Animated: Spinning core / flashing lights).
