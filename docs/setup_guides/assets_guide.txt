## 2. Audio Files
The `AudioManager` looks for specific file names. You can use `.ogg`, `.mp3`, or `.wav`.

| Sound Event | Expected Filename | Description |
|---|---|---|
| Background Music | `audio/music.mp3` | Loop during gameplay |
| Walk | `audio/walk.ogg` | (Not yet implemented in code loop, but prepared) |
| Attack | `audio/attack.ogg` | When player presses Space |
| Hit | `audio/hit.ogg` | When player or enemy takes damage |
| Kill | `audio/kill.ogg` | When enemy dies |
| Collect | `audio/collect.ogg` | When picking up Key |
| Victory | `audio/victory.ogg` | Win screen |
| GameOver | `audio/gameover.ogg` | Lose screen |

**Note**: If a file is missing, the game will purely skip the sound request. No crash.

## 3. Visual Effects
- **Red Flash**: Automatically handled by code (filters player texture red). No asset needed.
- **Particle Effects**: (Optional) You can add particle files in `particles/` folder later.

## 4. Key Bindings
Key bindings are stored in `~/.prefs/maze_runner_settings` (on Desktop) or local storage.
To reset to defaults, you can delete that file or implement a "Reset" button.
Current Defaults:
- Move: `UP`, `DOWN`, `LEFT`, `RIGHT` Arrows
- Attack: `SPACE`
